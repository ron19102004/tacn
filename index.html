Bits and Byte

@_Bit

The bit is the most basic unit of information computing and digital communications.

The bit represents a logical state with one of two possible values. These values are most commonly represented as either "1" or "0", but other representations such as true/false, yes/no, on/off, or +/− are also commonly used.

@_Parity_Bit

An extra bit at the beginning or end of a string of bits that is used to check if the data received is reasonably correct

• 8-bit ASCII has an extra bit added at the beginning of each 7-bit string.

• In even parity checking, the sum of all bits in the byte will be even and the value chosen for the eighth bit is selected based upon the sum of the other 7 bits.

• In odd parity checking, the sum of all bits in the byte will be odd and the value chosen for the eighth bit is selected based upon the sum of the other 7 bits.

Buying a computer

- If you want to purchase a computer, you must first choose between a desktop and a laptop.

- Each has advantages and disadvantages

-Desktops are typically more powerful.

  • They are stationary.

  • They are typically simpler to upgrade.

  • They require external components (keyboard, audio/visual components, etc.).

-With their portability, laptops often lose power.

  • Battery power varies strongly between models.

  • Upgrading components can be more difficult for some models.

  • Generally more expensive than comparable desktops.

  • All components included and enclosed.

- As you know, Now, the most popular operating systems are Windows and MacOS .Moreover, Linux is designed for advanced users.

Choosing an operating system is an important part of deciding to purchase a computer.

• Although Windows has the most users, apps, and support, but it’s viewed as the least safe and stable of the three.

• Linux can run most Windows programs and has the same basic requirements as Windows, but it has the least support options.

• MacOS is viewed as the most stable and is often used in "creative sectors."

Mouse button actions

Nowadays, every mouse includes at least two mouse buttons and may have additional buttons on the mouse's side.

@_Left mouse button

  1. This is used to click, select, or open an object on the computer. For example, you would left-click to open a link in a browser or double-click to open a program

  2. We can highlight text by holding down the left mouse button and dragging the mouse.

  3. In a spreadsheet, clicking a cell with the left mouse button makes it the active cell.

  4. In a game, the left button is used as the action button. For example, in a shooter game, the left button is often used to shoot a gun.

@_Right mouse button

  1. The right mouse button is used to give additional information, or the properties of a right-clicked item. For example, if you right-click anything highlighted, you are given a menu that lets you copy the highlighted text or another object.

@_Middle mouse button or wheel button

  1. Several computer mouse with wheels can be used as buttons as well. Pressing down on the mouse wheel functions like a button, and its action is determined by the software that came with the mouse. By default, the middle button opens a link in a new tab in the browser.
